# Project Log

Chronological record of all project interactions: calls, meetings, presentations, decisions.

## Naming Convention

Files follow the pattern: `YYYY-MM-DD_type_description.md`

Types:
- `call` — phone/video calls
- `meeting` — group meetings / workshops
- `presentation` — demos, presentations given
- `decision` — standalone decision records
- `note` — general notes, research findings
